# 웹서비스 설명

### 분신물 알림 서비스

>서울시 대중교통 분실물 습득물 정보 api 를 이용하여, 
>
>클라이언트가 등록한 분실물 정보가 공공api의 정보와 유사성이 있다면,
> 
>클라이언트의 이메일로 분실물의 정보(분실물명, 상태, 습득장소 등)를 발송한다.
---
# 기술 스택

* Java -version 11
* Gradle
* Spring Boot (API Server)
* Spring Security
* Spring Scheduler
* H2 Database
* JPA & QueryDSL
* JWT
* JUnit
* Google SMTP
